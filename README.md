Selamat datang ke repositori **Cryptorian World My** – sebuah **website statik** dengan **Luna AI Chatbot** pintar yang membantu pengguna di Malaysia memahami dan menggunakan **Worldcoin**, **Hata Wallet**, serta proses **Cashout WLD ke MYR**.

---

## 📌 TENTANG PROJEK INI

Ini adalah **website ringkas** yang mengandungi:
- ✅ **Panduan ringkas** Worldcoin, Hata Wallet & Cashout
- ✅ **Luna AI Chatbot** – AI comel yang boleh menjawab soalan berkaitan kripto
- ✅ **Butang salin** untuk kod rujukan `4RH0OTE` & `186300`
- ✅ **Reka bentuk moden** dengan Tailwind CSS & Heroicons

**Ini BUKAN bot Telegram.** Ini adalah **frontend website** yang boleh diakses terus melalui browser.

🔧 KONFIGURASI

Tukar AI Model

Dalam index.html, cari:

```javascript
const MODEL = "gemma-3-27b-it";
```

Boleh tukar ke model "gemini✨" jika mahu kepatuhan arahan lebih baik.

Tukar Kod Rujukan

Fungsi cariKodRujukan() dalam index.html mengandungi:

```javascript
const kodRasmi = ['4RH0OTE', '186300'];
```

Tambah atau buang kod mengikut keperluan.
